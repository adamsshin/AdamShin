2. 스윕스테이크 모델의 혁신과 진화

골드 코인 (GC)

  • • 게임 내 기본 가상화폐
  • • 현금 가치 없음
  • • 회원가입 시 무료 제공
  • • 추가 구매 가능
  • • 소셜 게임 전용

스윕스 코인 (SC)

  • • 프로모션용 토큰
  • • 무료 지급 (이벤트, 구매 혜택)
  • • 당첨 시 현금/상품 교환 가능
  • • 직접 판매하지 않음
  • • 스윕스테이크 모드 활성화

법적 포지셔닝 전략

VGW의 스윕스테이크 모델은 미국 도박법의 3요소 (금전적 대가, 우연, 상금) 중 "금전적 대가"를 회피하는 구조입니다. 사용자가 지불하는 돈은 가치 없는 골드 코인 구매이며, 스윕스 코인은 어디까지나 무료 경품이라는 논리로 "맥도날드 몬폴리 게임" 같은 프로모션과 동일하다고 주장합니다.
